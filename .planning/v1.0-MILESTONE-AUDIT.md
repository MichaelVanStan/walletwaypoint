## Critical Gaps (User-Facing)

### 1. 30 Dead Calculator Callout Links (HIGH)

All 10 calculator YAML configs contain 3 callout entries each with `href` values pointing to non-existent guide slugs. Every "related guide" card on every calculator page is a 404.

**Fix:** Update the 30 `href` values to reference actual guide slugs (`/guides/mortgage-payment`, `/guides/budget`, etc.)

### 2. Footer "Terms of Use" → 404 (HIGH)

`/terms` route does not exist. Footer links to it from every page.

**Fix:** Create `src/app/terms/page.tsx` or redirect to `/privacy-policy`

### 3. Calculator Pages Missing from Sitemap (MEDIUM)

10 `/calculators/[slug]` URLs are SSG-generated but not in `sitemap.xml`. Guides and hubs are properly listed.

**Fix:** Add calculator collection iteration to `src/app/sitemap.ts`

### 4. Calculator Navigation Disabled (MEDIUM)

Calculators section in header/footer has `disabled: true`. Users can only reach calculators via hub/guide links or direct URL.

**Fix:** Remove `disabled: true` from calculator entries in `src/lib/navigation.ts`
