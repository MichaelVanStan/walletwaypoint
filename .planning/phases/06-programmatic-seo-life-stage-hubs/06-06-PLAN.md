D-07 states: "Guide pages use hybrid approach — YAML for structured data plus editorial MDX sections."
This applies to hand-written editorial guides (e.g., those in `/guides/[slug]/`) which use MDX files with YAML frontmatter.

For these PROGRAMMATIC guide pages (state-taxes, homebuyer-programs), D-07's intent is satisfied differently:
- **Structured data** = YAML fields (brackets, programs, eligibility, rates) rendered by React components
- **Editorial content** = YAML `editorialContent` string fields containing per-state prose (300+ words)

These pages do NOT use MDX files because they are programmatic — the same React page component renders all 51 states from structured YAML data. The editorial content lives in the YAML `editorialContent`, `faqs`, and `tips` fields rather than in separate MDX files. This is the D-06 pattern (structured data templates) applied to guide-style content, which is the correct approach for programmatic SEO at scale (51 states from one page component).

Full MDX guide pages (per D-07's literal hybrid approach) are used for the hand-written hub guides in Plans 08 and 09.
</d07_clarification>

**1. Create 51 homebuyer program YAML files** in `content/homebuyer-programs/`.

**D-07 compliance note:** Same rationale as Task 1 — these are programmatic pages. The YAML `editorialContent` field provides per-state editorial prose. Structured program data (programs array with eligibility, amounts, agencies) is the YAML structured data component. Together they satisfy D-07's hybrid intent for programmatic guide pages.

Each file follows the Velite HomebuyerProgram schema. Data sourced from state Housing Finance Agency (HFA) websites, USAGov, and HUD-approved programs (per 06-RESEARCH.md data sourcing strategy).

Each state YAML includes:
- `slug`: state slug matching state-list.ts (e.g., "california")
- `stateName`: Full state name
- `stateAbbreviation`: Two-letter code
- `programs`: Array of 2-5 state-specific programs, each with:
  - `name`: Program name (e.g., "CalHFA FHA Loan Program")
  - `adminAgency`: Administering agency (e.g., "California Housing Finance Agency")
  - `assistanceType`: Type of assistance (e.g., "Down payment assistance", "Forgivable loan", "Grant")
  - `amount`: Dollar amount or range (e.g., "Up to 3.5% of purchase price")
  - `incomeLimits`: Income eligibility (e.g., "Must not exceed 150% of area median income")
  - `eligibleAreas`: Geographic scope (e.g., "Statewide" or specific counties)
  - `firstTimeBuyerRequired`: boolean
  - `url`: Link to official program page
- `editorialTitle`: "[State Name] First-Time Homebuyer Programs 2026"
- `editorialDescription`: Under 300 chars, unique per state
- `editorialContent`: 200+ words of state-specific content about homebuying in that state, housing market context, tips for first-time buyers
- `faqs`: 3-5 state-specific homebuyer FAQs
- `lastVerified`: "2026-03-30"
- `dataSource`: "State HFA / USAGov / HUD"

Also include federal programs available in every state:
- FHA loans (3.5% down)
- VA loans (0% down, eligible veterans only)
- USDA loans (0% down, rural areas)
- Good Neighbor Next Door (HUD)
