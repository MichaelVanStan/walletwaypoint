**1. Create `src/components/calculator/bracket-table.tsx`:**

A tabbed table component that displays state tax brackets by filing status. Per UI-SPEC:

Props interface:
```typescript
interface BracketTableProps {
  brackets: {
    single: Array<{ min: number; max: number; rate: number }>;
    married: Array<{ min: number; max: number; rate: number }>;
    head: Array<{ min: number; max: number; rate: number }>;
  };
  stateName: string;
  taxType: 'graduated' | 'flat' | 'none';
  flatRate?: number;
  hasIncomeTax: boolean;
}
```

Behavior:
- Uses shadcn Tabs component with 3 tabs: "Single", "Married Filing Jointly", "Head of Household"
- Each tab shows a shadcn Table with columns: "Income Range", "Tax Rate", "Tax on Bracket"
- Income Range format: "$0 - $11,079" using `formatCurrency` from calculators/formatters
- Tax Rate: "1.00%" right-aligned
- Tax on Bracket: calculated as `(max - min + 1) * rate / 100` for each bracket, right-aligned, currency format
- Top bracket shows "Over $X" instead of a range
- Table uses `text-sm` for cell text, `text-xs font-semibold` for headers
- Table header row uses `bg-muted/50` background (per UI-SPEC Secondary color)
- Cell padding: `px-2 py-2` (8px per spacing scale)
- Alternating row stripes via `even:bg-muted/30`

Special cases:
- **No-income-tax states (`hasIncomeTax === false`):** Render an informational card (shadcn Alert variant) instead of a table. Text: "[State] has no state income tax. Your entire salary is exempt from state income tax withholding." Followed by contextual content placeholder.
- **Flat-tax states (`taxType === 'flat'`):** Render a single-row table with the flat rate, plus explanatory text: "[State] uses a flat income tax rate of X.XX% on all taxable income."

Mark as `'use client'` only if needed for Tabs interaction — check if shadcn Tabs require client.
