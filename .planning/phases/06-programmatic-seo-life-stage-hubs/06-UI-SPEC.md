Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding, inline spacing |
| sm | 8px | Compact element spacing, table cell padding |
| md | 16px | Default element spacing, card padding, input gaps |
| lg | 24px | Section padding, form group spacing |
| xl | 32px | Layout gaps between input/result panels |
| 2xl | 48px | Major section breaks (charts to callouts) |
| 3xl | 64px | Page-level vertical spacing (hub hero to first section) |

Exceptions: Touch targets for the state indicator badge use 36px minimum height (9 x 4px) to ensure tappability on mobile while remaining visually compact. Bracket table cells use 8px vertical / 8px horizontal padding for dense data display.

## Interaction Contract

### Paycheck Calculator (`/calculators/paycheck`)

**Layout:** Standard two-column calculator layout (sticky input sidebar left, results right). Follows existing `CalculatorShell` pattern.

**Inputs (sidebar):**
- Annual Salary: currency slider+input (min: $15,000, max: $500,000, step: $1,000, default: $65,000)
- State: dropdown selector showing all 50 states + DC, grouped alphabetically. Selecting a state persists it via localStorage.
- Filing Status: radio group -- Single, Married Filing Jointly, Head of Household (default: Single)
- Pay Frequency: select -- Weekly, Biweekly, Semimonthly, Monthly (default: Biweekly)
- Additional Withholding: currency input (min: $0, max: $5,000, step: $10, default: $0)
- Pre-tax Deductions (401k, HSA): currency input (min: $0, max: $25,000/yr, step: $500, default: $0)

**Outputs (main panel):**
- Result scorecards: Net Pay Per Paycheck (primary), Annual Net Pay, Total Tax, Effective Tax Rate
- Pie chart: Tax breakdown (Federal, State, FICA, Take-Home) using `--chart-1` through `--chart-5`
- Bar chart: Monthly breakdown (Gross vs Net vs Taxes) stacked
- Interpretation text: "Based on your **$65,000** salary in **California**, your estimated take-home pay is **$X,XXX** per paycheck (biweekly), after **$X,XXX** in federal tax, **$X,XXX** in state tax, and **$X,XXX** in FICA."

**State page variant** (`/calculators/paycheck/[state]`):
- Same calculator inputs, but state is pre-selected and locked to the page's state
- Below calculator: BracketTable component showing state's tax brackets by filing status (tabs)
- Below brackets: 300+ words of editorial content (rendered from YAML `editorialContent` field)
- Below editorial: FAQ accordion (3-5 state-specific questions)
- Below FAQ: Tips callout cards (2-4 state-specific tips)
- Below tips: "Explore other states" link grid (neighboring states + popular states)

### Home Affordability Comparison Table (D-10)

**Layout:** Replaces the existing 6-scorecard grid with a single comparison table inside the results panel.

**Table structure:**
- 4 columns: Metric label | Conservative (28/36) | Moderate (30/43) | Aggressive (35/50)
- Column headers include the DTI rule as a subtitle: "28% housing / 36% total"
- Rows: Max Home Price, Monthly Payment, Housing DTI, Total DTI, Down Payment Amount, Estimated Monthly Disposable Income

**Row-specific formatting:**
- Max Home Price: currency, bold, largest visual weight in row
- Monthly Payment: currency
- Housing DTI: percent with DtiBadge color coding
- Total DTI: percent with DtiBadge color coding
- Down Payment Amount: currency
- Est. Monthly Disposable Income: currency (uses net income from persisted state; shows "Set your state for accurate estimate" link if no state persisted)

**Guidance banner:** Rendered below the table. Light blue-tinted background (accent/5). Text: "Lenders generally approve up to 43% total DTI. Below 36% gives you the most financial flexibility." Icon: Info circle (Lucide `Info`).

**Responsive behavior:** On screens below 640px (sm), the table scrolls horizontally inside its container. Column headers remain visible via sticky first column.

### State Indicator Badge (D-05)

**Placement:** Inside the header component, right-aligned before the mobile hamburger button. On desktop, visible inline in the navigation area. On mobile, visible in the sticky header bar.

**States:**
1. **No state selected (default):** Hidden entirely. No placeholder, no skeleton. This avoids hydration mismatch since the badge is a client component reading localStorage.
2. **State selected:** Compact badge showing state abbreviation. Format: "CA" in a small rounded badge with accent background. Badge dimensions: 32px wide x 28px tall. Text: 12px semibold, white on accent.
3. **Hover/click:** Opens a dropdown to change state. Dropdown lists all 50 states + DC alphabetically. Selecting a new state updates localStorage and all consuming components.

**Hydration strategy:** The badge renders `null` on the server and during initial client render. After `useEffect` reads localStorage, it fades in with `opacity` transition (150ms). This produces zero CLS because no space is reserved when hidden.

### Bracket Table Component

**Structure:** Tabbed interface (Filing Status tabs: Single | Married Filing Jointly | Head of Household). Each tab shows a table with columns:
- Income Range: "$0 - $11,079" format
- Tax Rate: "1.00%" with right alignment
- Tax on Bracket: calculated amount, right alignment

**For no-income-tax states:** Instead of a bracket table, show an informational card: "Alaska has no state income tax. Your entire salary is exempt from state income tax withholding." followed by contextual content about the state's other tax characteristics.

**For flat-tax states:** Show a single-row table with the flat rate, plus explanatory text about the flat tax structure.

## Accessibility Requirements

| Requirement | Implementation |
|-------------|----------------|
| Bracket table keyboard navigation | Standard table semantics. Tabs for filing status are keyboard-navigable via shadcn Tabs (arrow keys). |
| State selector | shadcn Select with proper aria-label "Select your state". Supports keyboard type-ahead. |
| DTI badge meaning | Each DtiBadge includes aria-label describing the status: "Housing DTI 28% -- within safe range" or "Total DTI 45% -- above recommended limit". |
| State indicator | aria-label "Your selected state is California. Click to change." When hidden, fully removed from DOM (not just visually hidden). |
| Color not sole indicator | DTI badges include text labels ("Safe", "Caution", "High") in addition to color coding. |
| Chart alternatives | Pie and bar charts include `aria-label` with text summary of data. Bracket data is in a proper table (screen reader accessible). |
| Reduced motion | Chart animations respect `prefers-reduced-motion` (existing pattern in `CalculatorShell`). |
