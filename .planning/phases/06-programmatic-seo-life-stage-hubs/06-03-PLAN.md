**1. Create `src/lib/states/state-persistence.ts`** — client-side hook for localStorage state persistence (per D-05):

```typescript
'use client';

import { useState, useEffect, useCallback } from 'react';
import type { PersistedState } from './types';

const STORAGE_KEY = 'ww_user_state';

export function usePersistedState() {
  const [userState, setUserState] = useState<PersistedState | null>(null);

  useEffect(() => {
    try {
      const stored = localStorage.getItem(STORAGE_KEY);
      if (stored) setUserState(JSON.parse(stored));
    } catch { /* SSR or localStorage unavailable */ }
  }, []);

  const setState = useCallback((state: PersistedState) => {
    setUserState(state);
    try {
      localStorage.setItem(STORAGE_KEY, JSON.stringify(state));
      // Dispatch storage event for cross-component sync
      window.dispatchEvent(new Event('storage'));
    } catch { /* quota exceeded or private browsing */ }
  }, []);

  const clearState = useCallback(() => {
    setUserState(null);
    try {
      localStorage.removeItem(STORAGE_KEY);
      window.dispatchEvent(new Event('storage'));
    } catch { /* ignore */ }
  }, []);

  return { userState, setState, clearState };
}
```

**2. Create `src/lib/calculators/math/paycheck.ts`** — paycheck calculator math module:

The function signature must match the existing calculator registry pattern: `(params: Record<string, number | string>) => CalculatorResults`.

Input params:
- `salary`: Annual gross salary (number)
- `state`: State slug (string, e.g., "california")
- `filing`: Filing status (string: "single" | "married" | "head")
- `frequency`: Pay frequency (string: "weekly" | "biweekly" | "semimonthly" | "monthly")
- `withholding`: Additional withholding per year (number, default 0)
- `pretaxDeductions`: Pre-tax 401k/HSA deductions per year (number, default 0)

Calculation steps:
1. Load state tax data from Velite: `import { states } from '#site/content'`
2. Compute taxable income: `salary - pretaxDeductions`
3. Compute federal tax using `calculateFederalTax(taxableIncome, filing)` from refactored tax.ts
4. Compute state tax:
   - If `taxType === 'none'`: state tax = 0
   - If `taxType === 'flat'`: state tax = taxableIncome * flatRate / 100
   - If `taxType === 'graduated'`: use `calculateProgressiveTax(taxableIncome - stateDeduction, stateBrackets[filing])`
   - Apply state standard deduction before bracket calculation (if state has one)
5. Compute FICA using `calculateFica()` (new function in this file):
   - Social Security: 6.2% on income up to $176,100 (2026 projected wage base)
   - Medicare: 1.45% on all income + additional 0.9% on income over $200,000
   - FICA applies to gross salary (not reduced by pre-tax deductions except for specific cafeteria plan deductions — for simplicity, apply to salary minus pretaxDeductions)
6. Total deductions = federal + state + FICA + additionalWithholding
7. Net pay = salary - pretaxDeductions - totalDeductions
8. Per-paycheck amounts based on frequency (weekly=52, biweekly=26, semimonthly=24, monthly=12)

Return `CalculatorResults` with:
- `outputs`: { netPayPerPaycheck, annualNetPay, totalTax, effectiveRate, federalTax, stateTax, socialSecurity, medicare, ficaTotal }
- `chartData.breakdown`: Pie chart with segments [Federal Tax, State Tax, FICA, Take-Home]
- `chartData.monthlyBreakdown`: Bar chart with [Gross, Taxes, Net] monthly
- `interpretation`: "Based on your **$65,000** salary in **California**, your estimated take-home pay is **$X,XXX** per paycheck (biweekly), after **$X,XXX** in federal tax, **$X,XXX** in state tax, and **$X,XXX** in FICA."
- `detailRows` / `detailColumns`: Breakdown table with federal, state, FICA line items

Export both `calculatePaycheck` and `calculateFica`.
