1. Create the page file with two parts:

**Server wrapper (top of file):**
- Export `metadata` using createMetadata: title "Financial Glossary", description "Plain-English definitions of common financial terms. No jargon, no fine print.", path "/glossary"

**Client Component `GlossaryContent` (in same file or separate):**
- Import `glossary` from `#site/content`
- State: `searchQuery` (string, default "")
- Filter terms: `glossary.terms.filter(t => t.term.toLowerCase().includes(searchQuery.toLowerCase()) || t.definition.toLowerCase().includes(searchQuery.toLowerCase()))`
- Group filtered terms by first letter: create a `Map<string, GlossaryTerm[]>` mapping A -> [APR, Amortization], C -> [Closing Costs, Compound Interest, Credit Score], etc.
- Sort letter groups alphabetically

**Layout (max-w-[720px] mx-auto px-4 sm:px-6 py-12 md:py-16):**
```
<h1 className="text-2xl font-semibold text-foreground">Financial Glossary</h1>
<p className="text-base text-muted-foreground mt-3">
  Plain-English definitions of common financial terms. No jargon, no fine print.
</p>

{/* Search input */}
<div className="mt-6">
  <Input
    type="search"
    placeholder="Search terms..."
    value={searchQuery}
    onChange={(e) => setSearchQuery(e.target.value)}
    className="max-w-sm"
  />
</div>

{/* A-Z jump links (optional horizontal bar) */}
<div className="mt-6 flex flex-wrap gap-2">
  {availableLetters.map((letter) => (
    <a key={letter} href={"#letter-" + letter}
       className="text-sm text-muted-foreground hover:text-foreground transition-colors">
      {letter}
    </a>
  ))}
</div>

{/* Letter groups */}
<div className="mt-8 space-y-8">
  {Array.from(groupedTerms.entries()).map(([letter, terms]) => (
    <section key={letter} id={"letter-" + letter}>
      <h2 className="text-xl font-semibold text-foreground mb-4 border-b border-border pb-2">
        {letter}
      </h2>
      <dl className="space-y-4">
        {terms.map((term) => (
          <div key={term.slug} id={"term-" + term.slug}>
            <dt className="font-semibold text-foreground">{term.term}</dt>
            <dd className="text-base text-muted-foreground mt-1">
              {term.longDefinition || term.definition}
            </dd>
          </div>
        ))}
      </dl>
    </section>
  ))}
</div>

{/* Empty search state */}
{filteredTerms.length === 0 && searchQuery && (
  <p className="text-sm text-muted-foreground mt-8">
    No terms match "{searchQuery}". Try a different search or browse the full list below.
  </p>
)}
```

- Use `<dl>` (definition list) for semantic HTML -- appropriate for glossary terms
- Letter group headings use H2 (text-xl font-semibold per UI-SPEC Typography)
- Separator between letter groups via `border-b border-border pb-2` on the H2
- Smooth scroll behavior: add `scroll-behavior: smooth` is already set globally or use `scroll-margin-top` on letter sections
- Add `scroll-margin-top: 80px` (or `scroll-mt-20`) to letter section headings so they don't scroll behind the sticky header
