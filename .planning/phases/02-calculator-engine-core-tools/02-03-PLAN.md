**Implementation:**

The component iterates over the `charts` array and renders the appropriate Recharts chart for each entry. Each chart is wrapped in a `<div>` with a title heading (`text-sm font-semibold text-foreground mb-3`).

**Chart type rendering:**

1. **AreaChart** (for amortization curves, growth projections, cumulative cost):
   - Import: `ResponsiveContainer, AreaChart, Area, XAxis, YAxis, Tooltip, Legend` from recharts
   - Height: 300px on desktop (lg+), 240px on mobile (use responsive container with fixed height via parent div: `h-[240px] md:h-[300px]`)
   - Gradient fills: Define `<defs>` with `<linearGradient>` for each data series using CSS custom property values. Since Recharts SVG may not support CSS custom properties directly, use the OKLCH literal values:
     - Series 1 gradient: `oklch(0.35 0.1 245)` at 30% opacity top, 0% at bottom (id: `gradient-1`)
     - Series 2 gradient: `oklch(0.55 0.12 175)` at 30% opacity top, 0% at bottom (id: `gradient-2`)
   - Area: `type="monotone"`, `strokeWidth={2}`, `fill="url(#gradient-N)"`, `stroke` using OKLCH values
   - XAxis: `tick={{ fontSize: 14 }}`, no gridlines
   - YAxis: `tickFormatter` using formatCurrency for currency axes (divide by 1000 and append "K" for large values)
   - Tooltip: custom formatter using formatCurrency/formatNumber, background `hsl(var(--card))` with border `hsl(var(--border))` -- actually use the OKLCH values from globals.css as fallback since Recharts tooltip uses inline styles
   - No gridlines: `<CartesianGrid>` is NOT rendered (per D-16: no gridlines)
   - Animation: `animationDuration={reducedMotion ? 0 : 300}`, `isAnimationActive={!reducedMotion}` (per UI-SPEC)
   - Detect series keys dynamically from the first data point (exclude the X-axis key which is typically 'year', 'month', 'age', or 'label')

2. **PieChart** (for breakdowns):
   - Import: `ResponsiveContainer, PieChart, Pie, Cell, Tooltip, Legend` from recharts
   - Same responsive height as AreaChart
   - Colors: cycle through chart-1 through chart-5 OKLCH values for cells
   - `innerRadius={60}` for donut style, `outerRadius={100}`
   - Tooltip: show label + formatted value
   - Legend: below chart, `verticalAlign="bottom"`, `iconType="circle"`
   - Animation: same reduced motion handling

3. **LineChart** (for savings progress, projections):
   - Import: `ResponsiveContainer, LineChart, Line, XAxis, YAxis, Tooltip, Legend, ReferenceLine` from recharts
   - Same responsive height
   - `strokeWidth={2}`, no dots by default (`dot={false}`)
   - Colors: chart-1 for primary series, chart-2 for secondary
   - Optional ReferenceLine for goal targets (if data contains a `goal` field)
   - No gridlines, same axis formatting as AreaChart

4. **BarChart** (for plan comparisons, bracket visualization):
   - Import: `ResponsiveContainer, BarChart, Bar, XAxis, YAxis, Tooltip, Legend, Cell` from recharts
   - Same responsive height
   - `barSize={40}` for readable bars
   - Colors: chart-1 through chart-5 cycling
   - Rounded bar caps: `radius={[4, 4, 0, 0]}`
   - No gridlines

```typescript
function CustomTooltip({ active, payload, formatter }: any) {
```

```typescript
/** Download data as CSV file with UTF-8 BOM for Excel compatibility */
export function downloadCsv(filename: string, headers: string[], rows: (string | number)[][]): void {
  const bom = '\uFEFF';
  const csvContent = bom + [
    headers.join(','),
    ...rows.map(row => row.map(cell =>
      typeof cell === 'string' && (cell.includes(',') || cell.includes('"') || cell.includes('\n'))
        ? `"${cell.replace(/"/g, '""')}"`
        : cell
    ).join(','))
  ].join('\n');

  const blob = new Blob([csvContent], { type: 'text/csv;charset=utf-8;' });
  const url = URL.createObjectURL(blob);
  const link = document.createElement('a');
  link.href = url;
  link.download = `${filename}.csv`;
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
  URL.revokeObjectURL(url);
}
```
